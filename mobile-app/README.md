# Mobile App Documentation

## Overview
This mobile application is built using Node.js and Expo, providing a seamless user experience with authentication features using Gmail and Apple ID.

## Project Structure
```
mobile-app
├── src
│   ├── App.js
│   ├── components
│   │   └── LoginButton.js
│   ├── screens
│   │   └── LoginScreen.js
│   ├── services
│   │   ├── authService.js
│   │   └── api.js
│   └── navigation
│       └── AppNavigator.js
├── package.json
├── app.json
└── README.md
```

## Features
- User authentication via Gmail and Apple ID.
- Navigation setup for different screens.
- Modular structure for components and services.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd mobile-app
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Usage
- Launch the app on your mobile device or emulator.
- Navigate to the login screen to authenticate using Gmail or Apple ID.
